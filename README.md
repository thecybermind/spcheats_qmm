This plugin only works with Quake 2, Quake 2 Remaster, and Quake 3 only!

1. Install QMM ( https://github.com/thecybermind/qmm2/wiki/Installation )
2. Make a qmmaddons/spcheats directory inside your mod directory and place spcheats_qmm.dll here
3. Add the path to spcheats_qmm.dll as an entry in the plugins list in qmm2.json
